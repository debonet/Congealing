Detailed Description

template<class DATA, int DIMENSIONALITY, class PRECISION, class SOURCE>
class SourceTransformOf< DATA, DIMENSIONALITY, PRECISION, SOURCE >

Core of all transform and accessor sources SourceTransforms (and deriveds) Defines the dimension, type and precision dependent parts of the SourceTransform interface

Template Parameters:
DATA the storage type of the source
DIMENSIONALITY the number of dimensions of the source
PRECISION the class used to index each dimension
SOURCE the class of the underlying source which is transformed by this layer
